# Todo App

A simple, interactive todo application that helps users manage their daily tasks. The Todo App allows users to:

Add New Todos: Users can click a button to open a form where they enter a task name and optional due date.

Mark Todos As Complete: Each todo has a checkbox that users can check/uncheck.

Delete Todos: Each todo has a delete button to remove it.

View All Todos: Todos are displayed in a list format showing the task name and due date.

Form Validation: The form validates that users enter requiered information before submitting.

## Functionality

CRUD Operations: Creating new tasks, reading/displaying lists, updating task status, and deleting tasks.

Task Management: Marking tasks as completed/incomplete, with visual indicators.

Data Handling: Storing and retrieving task data, frequently utilizing technologies like JavaScript.

User Interface (UI): Creating interactive elements for task input and management.

Git Version Control: Using Git branches and pull requests to manage code development.

Filtering: Filtering tasks by status (e.g., Active, Completed, All).

## Technology

HTML: Structure and layout.

CSS: Styling and responsive design.

JavaScript: Interactive functionality.

Object-Oriented Programming: Todo and FormValidator classes.

DOM Manipulation: Adding, removing, and updating tools.

Form Validation: Real-time input validation.

UUID Generation: Unique IDs for each todo.
